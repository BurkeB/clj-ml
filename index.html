An Overview of the clj-ml API


clj-ml is a Clojure-based, machine learning library built on top of the Weka framework for the JVM.


classifiers

Detailed API documentation 
        This namespace contains several functions for
        building classifiers using different classification algorithms: Bayes
        networks, multilayer perceptron, decission tree or support vector machines are
        available. Some of these classifiers have incremental versions so they can be
        built without having all the dataset instances in memory. Functions for
        evaluating the classifiers built using cross validation or a training set are
        also provided
Public variables and functions: classifier-classify classifier-evaluate classifier-label classifier-train classifier-update make-classifier make-classifier-options

clusterers

Detailed API documentation
      This namespace contains several functions for
      building clusterers using different clustering algorithms. K-means, Cobweb and
      Expectation maximization algorithms are currently supported. Some of these
      algorithms support incremental building of the clustering without having the
      full data set in main memory. Functions for evaluating the clusterer as well
      as for clustering new instances are also supported
Public variables and functions: clusterer-build clusterer-cluster clusterer-evaluate clusterer-info clusterer-update make-clusterer make-clusterer-options

data

Detailed API documentation
      This namespace contains several functions for
      building creating and manipulating data sets and instances. The formats of
      these data sets as well as their classes can be modified and assigned to
      the instances. Finally data sets can be transformed into Clojure sequences
      that can be transformed using usual Clojure functions like map, reduce, etc.
Public variables and functions: attribute-name-at index-attr instance-set-class instance-to-map instance-to-vector instance-value-at is-dataset? is-instance? make-instance make-dataset dataset-add dataset-at dataset-class dataset-class-value dataset-count dataset-extract-at dataset-format instance-get-class dataset-name dataset-pop dataset-seq dataset-values-at

data-store

Detailed API documentation 
        Functions for storing and retrieving data
        sets from a persistence store, like a data base system. Currently
        MongoDB is the only store supported.
Public variables and functions: data-store-connection-db data-store-load-dataset data-store-save-dataset make-data-store-connection

distance-functions

Detailed API documentation 
        Generates different distance metrics that can
        be passed as parameters to certain classifiers and clusterers like
        K-Means. Euclidean, Manhattan and Chebysev distance functions are supported.
Public variables and functions: make-distance-function

filters

Detailed API documentation 
        Functions in this namespace define transformations that can be applied
        to any loaded data set.
Public variables and functions: make-filter filter-apply make-apply-filter

io

Detailed API documentation 
        Functions for reading and saving datasets, classifiers and clusterers to
        files and other persistence mechanisms.
Public variables and functions: load-instances save-instances

kernel-functions

Detailed API documentation 
        Kernel functions that can be passed as parameters to support vector
        machines classifiers. Polynomic, radial basis and string kernels are supported.
Public variables and functions: make-kernel-function

ui

Detailed API documentation 
        Namespace containing functions for plotting classifiers, clusterers and
        data sets.
Public variables and functions: visualiza-plot display-object datase-display-numeric-attributes clusterer-display-for-attributes
